## 📖 Overview
This repository contains the deliverables for **Task 1** of the **ApexPlanet Data Analytics Internship**. The primary focus of this project is to clean, transform, and prepare a raw sales dataset for comprehensive exploratory data analysis and visualization.

## 🎯 Objective
Clean and prepare a sales dataset for analysis through data profiling, handling missing values, identifying duplicates, and ensuring data quality across all features.

---

## 📂 Repository Structure

```text
Task1_Data_Analytics/
│
├── datacleaning.py                       # Python script for data wrangling
├── Data_Dictionary.md                    # Comprehensive documentation of dataset features
├── README.md                             # Project documentation (You are here)
├── ApexPlanet_DataAnalytics_Dataset.xlsx # Original raw dataset
└── Cleaned_Sales_Dataset.csv             # Cleaned dataset ready for downstream analysis
```

---

## 🛠️ Tasks Performed
- **Data Profiling:** Initial assessment of dataset dimensions and data types.
- **Missing Value Handling:** Identifying and imputing/dropping missing values.
- **Duplicate Checking:** Removing redundant rows to maintain data integrity.
- **Date Conversion:** Parsing and standardizing date formats.
- **Outlier Detection:** Identifying anomalies in numerical columns.

---

## 📈 Outcome
Successfully transformed the raw `ApexPlanet_DataAnalytics_Dataset.xlsx` into `Cleaned_Sales_Dataset.csv`, generating a clean, structured dataset ready for deep-dive statistical analysis and dashboard creation in subsequent tasks.
